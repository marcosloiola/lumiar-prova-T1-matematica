1. Adição com Números Inteiros
🔹 Quando os sinais são iguais:
Soma os módulos (valores absolutos) dos números e mantém o sinal.
Exemplos:
(+4) + (+5) = +9
(–2) + (–3) = –5
(+10) + (+8) = +18
(–7) + (–6) = –13
Entendendo:
Se você deve R$5 e depois deve mais R$3, no total deve R$8. (-5) + (-3) = -8
🔹 Quando os sinais são diferentes:
Subtrai os módulos (maior módulo menos o menor) e mantém o sinal do número com maior módulo.
Exemplos:
(+25) + (–5) = +20 (|25| > |5|, então sinal positivo)
(–36) + (+10) = –26 (|36| > |10|, então sinal negativo)
(+8) + (–12) = –4 (|12| > |8|, então sinal negativo)
(–15) + (+40) = +25 (|40| > |15|, então sinal positivo)
Para visualizar melhor: imagine uma reta numérica. Números positivos vão para a direita, negativos para a esquerda. Você começa do zero e segue na direção indicada.
2. Propriedades da Adição
Comutativa

Você pode trocar a ordem dos números que o resultado não muda.

a + b = b + a

Ex: (+3) + (–2) = (–2) + (+3) = +1

Associativa

Você pode mudar a ordem de agrupamento sem alterar o resultado.

(a + b) + c = a + (b + c)

Ex: [(+2) + (–3)] + (+5) = (+2) + [(–3) + (+5)]

Elemento Neutro

O zero é o elemento neutro da adição. Qualquer número somado com zero é igual a si mesmo.

a + 0 = a

Ex: (–7) + 0 = –7

Elemento Oposto

Todo número inteiro possui um oposto (inverso aditivo). Sua soma resulta em zero.

a + (–a) = 0

Ex: (+5) + (–5) = 0

3. Subtração com Números Inteiros
Conceito Fundamental:
Subtrair é o mesmo que somar o oposto!
a – b = a + (–b)

Para subtrair, transforme a subtração em uma adição do oposto do segundo número.
Exemplos Passo a Passo:
(+31) – (+26) = (+31) + (–26) = +5
(–2) – (–6) = (–2) + (+6) = +4
(+8) – (–6) = (+8) + (+6) = +14
(–10) – (+3) = (–10) + (–3) = –13
(+15) – (–8) = (+15) + (+8) = +23
Observe o padrão: quando você subtrai um número negativo, o sinal dele muda para positivo! Isso é porque o oposto de um número negativo é positivo.
4. Multiplicação com Números Inteiros
🔹 Regra dos Sinais na Multiplicação:
Lembre-se:
Sinais iguais → resultado POSITIVO (+)
Sinais diferentes → resultado NEGATIVO (–)
Exemplos:
(+3) · (+2) = +6 (positivo com positivo = positivo)
(–3) · (–2) = +6 (negativo com negativo = positivo)
(+3) · (–2) = –6 (positivo com negativo = negativo)
(–3) · (+2) = –6 (negativo com positivo = negativo)
🔹 Propriedades da Multiplicação:
Comutativa
a · b = b · a

Ex: (–3) · (+2) = (+2) · (–3) = –6

Associativa
(a · b) · c = a · (b · c)

Ex: [(–2) · (+3)] · (+4) = (–2) · [(+3) · (+4)]

Elemento Neutro
a · 1 = a

Ex: (–5) · 1 = –5

Elemento Nulo
a · 0 = 0

Ex: (+8) · 0 = 0

Distributiva
a · (b + c) = a·b + a·c

Ex: (–2) · [(+3) + (–4)] = (–2)·(+3) + (–2)·(–4) = –6 + 8 = 2

Multiplicação de 3 Números

Se houver um número par de fatores negativos, o resultado é positivo. Se ímpar, é negativo.

Ex: (–2) · (–3) · (+4) = +24 (2 negativos = par)

5. Divisão com Números Inteiros
🔹 Regra dos Sinais na Divisão:
A regra é a mesma da multiplicação!
Sinais iguais → quociente POSITIVO (+)
Sinais diferentes → quociente NEGATIVO (–)
Exemplos:
(–42) ÷ (–3) = +14 (sinais iguais = positivo)
(+450) ÷ (–30) = –15 (sinais diferentes = negativo)
(+64) ÷ (+8) = +8 (sinais iguais = positivo)
(–100) ÷ (+5) = –20 (sinais diferentes = negativo)
A divisão é a operação inversa da multiplicação. Se você tem dúvida em uma divisão, pode verificar multiplicando o resultado pelo divisor!
6. Potenciação com Números Inteiros
🔹 O que é uma potência?
aⁿ = a · a · a · ... · a (n vezes)

Onde a é a base (número que se repete) e n é o expoente (quantas vezes se repete).

🔹 Expoente Par vs Expoente Ímpar:
Expoente PAR

Sempre resulta em número POSITIVO, independente do sinal da base!

Ex: (–3)⁴ = (–3)·(–3)·(–3)·(–3) = +81

Ex: (–2)² = (–2)·(–2) = +4

Expoente ÍMPAR

O resultado tem o mesmo sinal da base!

Ex: (–3)⁵ = (–3)·(–3)·(–3)·(–3)·(–3) = –243

Ex: (+2)³ = (+2)·(+2)·(+2) = +8

🔹 Propriedades da Potenciação:
1. Multiplicação de Potências com Mesma Base:
aᵐ · aⁿ = aᵐ⁺ⁿ

Ex: (–2)³ · (–2)² = (–2)⁵ = –32

2. Divisão de Potências com Mesma Base:
aᵐ ÷ aⁿ = aᵐ⁻ⁿ

Ex: (+3)⁵ ÷ (+3)² = (+3)³ = +27

3. Potência de Potência:
(aᵐ)ⁿ = aᵐ·ⁿ

Ex: [(–2)²]³ = (–2)⁶ = +64

4. Expoente Zero:
a⁰ = 1 (para qualquer a ≠ 0)

Ex: (–5)⁰ = 1, (+100)⁰ = 1

5. Expoente Um:
a¹ = a

Ex: (–8)¹ = –8, (+12)¹ = +12

7. Radiciação com Números Inteiros
🔹 O que é uma raiz?
ⁿ√a (raiz enésima de a)

A raiz é a operação inversa da potenciação. Se 2³ = 8, então ³√8 = 2

🔹 Raiz Quadrada de Negativos:
Importante!
Raiz quadrada de número negativo NÃO existe em ℤ (no conjunto dos números inteiros)

√(–4) não existe em ℤ porque nenhum número inteiro elevado ao quadrado resulta em um número negativo.
🔹 Raiz Cúbica de Negativos:
Raiz cúbica de número negativo EXISTE!
Exemplos:
³√(–27) = –3 porque (–3)³ = –27
³√(–8) = –2 porque (–2)³ = –8
³√(–125) = –5 porque (–5)³ = –125
Resumindo: raízes de índice ÍMPAR (como raiz cúbica) podem ter resultado negativo. Raízes de índice PAR (como raiz quadrada) não podem ter números negativos dentro delas.
8. Expressões Numéricas
🔹 Ordem de Operações (Prioridades):
Potenciação e Radiciação (do maior para o menor índice/expoente)
Multiplicação e Divisão (da esquerda para a direita)
Adição e Subtração (da esquerda para a direita)
Símbolos de Agrupamento:
Parênteses ( ) → Colchetes [ ] → Chaves { }

Resolva sempre o que está dentro desses símbolos, começando pelos parênteses!
Exemplo Completo:

–2 + 3 · (–4)² ÷ 2 – 5

Passo 1: Potenciação → (–4)² = +16

–2 + 3 · 16 ÷ 2 – 5

Passo 2: Multiplicação e divisão (esquerda para direita)

3 · 16 = 48 → 48 ÷ 2 = 24

–2 + 24 – 5

Passo 3: Adição e subtração (esquerda para direita)

–2 + 24 = 22 → 22 – 5 = +17

Outro Exemplo com Parênteses:

[(+2) – (–3)] · (–4) + 5

Passo 1: Parênteses

(+2) – (–3) = (+2) + (+3) = +5

[+5] · (–4) + 5

Passo 2: Multiplicação

+5 · (–4) = –20

–20 + 5 = –15

Resumo das Regras Principais
✓ Adição: Sinais iguais = soma módulos
✓ Subtração: a – b = a + (–b)
✓ Multiplicação: Sinais iguais = positivo
✓ Divisão: Mesma regra que multiplicação
✓ Potenciação: Par = positivo; Ímpar = sinal da base
✓ Ordem: Potência → Mult/Div → Ad/Sub
